# 📱 Lattes de Bolso

Aplicativo mobile desenvolvido em React Native com Expo, inspirado na plataforma Lattes do CNPq. Reúne informações acadêmicas e profissionais de forma simples e acessível.

## 🔧 Tecnologias

- React Native
- Expo
- React Navigation (Stack Navigator)
- Expo Vector Icons

## 📄 Telas

- **Landing Page** — tela de entrada com estilo terminal
- **Home** — menu principal de navegação
- **Perfil** — informações pessoais e de contato
- **Formação** — histórico acadêmico
- **Cursos** — cursos realizados
- **Conhecimentos** — habilidades e tecnologias
- **Links** — redes sociais e contatos

## 🚀 Como rodar

```bash
# Instalar dependências
npm install

# Iniciar o projeto
npx expo start
```
